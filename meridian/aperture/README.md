# The Meridian Engine: Aperture — HTML Test

A standalone browser vertical slice for **Nahar Meridian / Project High Sun**.

## Features

- First-person movement inside the Aperture test chamber
- Mouse, keyboard, browser gamepad, and touchscreen input
- Three physical alignment consoles
- Three independently rotating solar rings
- Light-routing beams and a powered artificial sun
- Local Light Memory completion record
- **FIRST LIGHT** Mark and return to the Meridian Horizon

## Controls

- WASD or left stick: move
- Mouse or right stick: look
- E, Enter, Space, or controller south button: interact / Sun
- Escape or controller east button: leave console / Door
- At an alignment console, Left/Right rotates the ring and Sun attempts to lock it

This test is fully isolated inside `meridian/aperture/` and has no dependency on the Pico project.
