# Daybreak OS Meridian — Horizon Prototype

An interactive browser prototype for **Nahar Meridian NMC-01**, the more powerful home-console companion to Nahar Pico.

## Included

- Cinematic High Sun boot sequence
- The Horizon world carousel
- Five launch worlds:
  - The Meridian Engine
  - Mothwell: Below
  - Black Falcon: Ash Sky
  - Dune Signal: Station Nine
  - Door Zero
- Archive, Sessions, Light Memory, Observatory, Signal, and Workshop
- Meridian system overlay and simulated power states
- World Key insertion simulation
- Nahar Pico connection simulation
- Persistent Marks, settings, selection, and fake download progress
- Keyboard, browser gamepad, mouse, and touchscreen support
- Five interactive Canvas world demonstrations
- Reduced-motion, high-contrast, and text-scale options

## Keyboard controls

- **Left / Right:** Move across worlds or options
- **Down:** Open system spaces
- **Enter / Space / A:** Sun — play, confirm, or act
- **Escape / B:** Door — back or exit
- **X:** Signal — contextual options
- **Y:** Mark — open Observatory
- **Q / G:** Meridian Guide overlay
- **K:** Simulate the selected World Key
- **P:** Connect a simulated Nahar Pico

## Touch

Touch controls are enabled automatically on coarse-pointer devices and can be shown or hidden from the top-right **Touch Controls** button.
