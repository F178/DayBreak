# Daybreak OS Web Showcase

A fully interactive browser mockup of **Daybreak OS** for the fictional **Nahar Pico NPC-01** micro-console by Nahar Works.

## Features

- Animated Daybreak OS boot sequence
- Shelf-style console dashboard
- Five interactive mini-games: Mothwell, Dune Signal, Black Falcon, 100 Doors, and Coin Saint
- Light Memory high scores using browser localStorage
- Settings and system diagnostics
- Keyboard, mouse, and touch controls
- Fullscreen support and synthesized UI/game sounds
- Responsive desktop and mobile layout

## Run locally

Open `index.html` directly, or serve the folder with any static server:

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Controls

- Dashboard: Arrow keys, Enter/A, B/Escape, F for fullscreen
- Games: Arrow keys plus A/B; P pauses; M or Escape returns to Shelf

All visual assets and the interface are original to Project Small Sun / Nahar Works.
